#  Hierarchy Builder (BFHL Full Stack Project)

A full-stack web application that processes node relationships, builds hierarchical trees, detects cycles, and returns structured insights via a REST API with a simple frontend interface.

##  Problem Statement

Build a REST API that:
- Accepts node relationships (e.g., A->B)
- Constructs hierarchical tree structures
- Detects cycles in the graph
- Returns structured output
